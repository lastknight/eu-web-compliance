# Interview kit

The three tier client questionnaire (gates, prefilled, detail) that turns 200+ potential questions into a 20 to 40 question conversation. See "The method in five minutes" in the root README.

The questionnaire document is being finalized and lands here shortly. Question IDs (`Q-{area}-{nn}`) are the exchange format between Run 1 (scan + prefill) and Run 2 (verification): do not rename them when filling in answers.
